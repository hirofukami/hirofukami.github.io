Gmail

Gmailを使って結構たつが、かなり便利だなぁと思う今日この頃。

メインのメアドに切り替え中。

  • コマンドラインが使える
  • フィルタがかけられ、フォルダ分けのようなことも出来る # MS Outlook Express 並
  • SPAMを自動的にフォルダに分けてくれる
  • インターフェイスが Web なので、端末を問わない # 当たり前か、、、

という点で良いです。

50個まで紹介できるので誰かほしかったらお知らせしてちょ。

＃ といいつつ、紹介したけどカウントは一時的に減って、また50になった。

＃ 永久不滅ポイントですか？
