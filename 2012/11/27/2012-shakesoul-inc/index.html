2012年 自分の会社(ShakeSoul Inc.)でのお仕事リスト

インフラエンジニア

  • IaaS: AWS、GMOアプリクラウド
  • DNSサーバ構築 : PowerDNS(Master/Slave構成) + Poweradmin
  • DNSキャッシュサーバ検証 : Unbound
  • 監視サーバ構築 : Pandora FMS、ZABBIX
  • Heroku パフォーマンス調査、改善検証
  • Private Cloud 検証 : CloudStack 3.0.2 + Xenserver/KVM/
  • Cloud Manage : RightScale
  • NextIO パフォーマンス測定検証(Fusion-ioを搭載した外付けストレージ)
  • ネットワーク : アライドL2スイッチ、Cisco Catalyst L3スイッチ

コンサルティング

  • ミーティング

    • スタンドアップミーティング
    • すごい会議の手法 : 報告、リクエスト、質問にもとづいて会話を進める
    • 究極の会議の手法 : 議事録ドリブン、プロジェクターにテキストエディタで記述、TODOを作ってサイクルさせる
  • ドキュメント改善
    • ワード、エクセルベースから編集が容易で共有しやすいWikiベースに
  • チケットベースの業務オペレーション
    • やったことがすべてチケットに載っているように

ネットワーク設計・検証

  • Single Point of Failure のないネットワーク構成設計
  • L3スイッチ Catalyst 3650
  • L2スイッチ Catalyst 2690
  • Firewall Palo Alto
  • Load Balancer A10 Networks
  • すべてのポイントをダウンさせて、10秒以内の切り替り検証

MBaaS(Mobile Backend as a Service) KiiCloud の日本でのプロモーション支援

  • プロモーションツール作成
  • サービスオペレーション測定方法定義
  • 起業を目指している人たちとの交流
  • 国内データセンタなどのパートナーシップづくり

前半はエンジニア的な業務内容もやりながら、業務コンサルティングのような幅のあるノウハウと俯瞰した視点が必要なものもやった。後半は一転してサービスプロモーション支援もやるようになった。

こうやってみるとかなり仕事の幅が広い。
これらをやりながら iPhone アプリも作ってきたのだから、一人の人間の中で結構カオスな状態になっていて、なかなか面白い。このカオスな状態でそこそこのパフォーマンスを出せるのが自分の特徴だと思うので、これはこれで気に入っていたりする。

さて、来年はどんなふうに振舞っていこうか。
